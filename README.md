# 🏏 IPL Winners Dashboard (2008-2025)

## 📌 Project Overview
This project is an interactive Microsoft Excel dashboard that analyzes IPL data from 2008 to 2025. The dashboard provides insights into winners, runner-up teams, player awards, match trends, and venue statistics.

## 📊 Dashboard Highlights
- Season-wise IPL Winners and Runner-up Teams
- Player of the Season Analysis
- Match Wins based on Bat First vs Bowl First
- Toss Decision Winning Percentage
- Top 10 Venues Analysis
- Team-wise Trophy Distribution
- Man of the Match Leaders
- Interactive Slicers and Dynamic KPI Cards
